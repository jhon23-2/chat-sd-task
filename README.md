# 💬 Chat SD Task

A real-time chat application built with **Node.js**, **Express**, **Socket.IO** and **React + Vite**.

## 🚀 Live Demo

🔗 [https://chat-sd-task.onrender.com](https://chat-sd-task.onrender.com)

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite |
| Backend | Node.js + Express |
| Real-time | Socket.IO |
| Deployment | Render.com |

## 📁 Project Structure

```
chat-sd-task/
├── client/   → React frontend (Vite)
└── server/   → Express + Socket.IO backend
```

## ▶️ Run locally

```bash
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm run dev
```

## ✨ Features

- Real-time messaging with Socket.IO
- Shows online users
- System notifications when users join or leave
- Single deployment — server serves the React build
